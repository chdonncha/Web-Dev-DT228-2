ABOUT


Don's Book Reservations is here to provide quality, user friendly and easy to use book reservations. This website was created as part of an assignment for the Website Development course in DIT.

The aim of the website is to provide facilities using PHP and HTML to allow users to sign in or register, then been able to either browse or reserve books with once a book been reserved it would be linked to the users accounts, then allowing them to view their reserved books and allowing the user to un-reserve their book.
